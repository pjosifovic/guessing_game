Guessing game description

What JS does on this page

Receives user input through a 'prompt' window that asks “What is your name?” and stores that input into a variable.

The inputed value is logged into the Google Chrome console using the debbuging message.

JS will return message via alert window saying 'Welcome, user's name, to our guessing game.

Aftet that user will receive three questions in the 'prompt'window and he/she will respond in the 'alert' window using 'Yes' or 'No', capitalize.

If the anwser is wrong, alert window will pop up displaying correct answer.

When the questions are finished, message will be displayed telling how many questions user got correct and tells which one they missed with the correct answer
